# Changes

## 1.0.0

Initial public release under the `Executables` package name.

Highlights:

- executable and executor abstractions promoted to the center of the library model,
- execution operators added,
- policies and maps added,
- policy handlers deprecated,
- direct handler-based data transformation deprecated in favor of executable maps,
- executable and handleable interfaces extracted,
- event handling model revised,
- multiple bug fixes and API refinements.
